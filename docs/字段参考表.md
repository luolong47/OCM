# OpenCode config.json 完整字段参考表

## 根级配置字段

| 字段名 | 类型 | 默认值 | 取值范围 | 说明 |
|--------|------|--------|----------|------|
| `$schema` | string | - | 有效的 URL | JSON Schema 引用，用于配置验证 |
| `shell` | string | 系统默认 Shell | 任何有效的 Shell 路径（如 `/bin/bash`, `/bin/zsh`） | 默认 Shell，用于终端和 bash 工具 |
| `username` | string | 系统用户名 | 任意字符串 | 对话中显示的自定义用户名 |
| `logLevel` | string | `"INFO"` | `"DEBUG"`, `"INFO"`, `"WARN"`, `"ERROR"` | 日志输出级别 |
| `snapshot` | boolean | `true` | `true`, `false` | 是否启用文件系统快照（用于撤销/重做） |
| `share` | string | `"manual"` | `"manual"`, `"auto"`, `"disabled"` | 会话分享控制模式 |
| `autoupdate` | boolean/string | `true` | `true`, `false`, `"notify"` | 自动更新控制，`notify` 仅通知不更新 |
| `model` | string | - | `"提供商/模型名"` 格式 | 主要使用的 AI 模型 |
| `small_model` | string | - | `"提供商/模型名"` 格式 | 用于简单任务的小模型（如标题生成） |
| `default_agent` | string | `"build"` | 已定义的 agent 名称 | 默认使用的智能体，必须是 primary 类型 |
| `instructions` | array | `[]` | 字符串数组（文件路径或模式） | 额外的指令文件或匹配模式 |
| `disabled_providers` | array | `[]` | 提供商 ID 字符串数组 | 禁用的模型提供商列表 |
| `enabled_providers` | array | `[]` | 提供商 ID 字符串数组 | 仅启用的提供商列表（白名单） |

## agent 智能体配置字段

| 字段名 | 类型 | 默认值 | 取值范围 | 说明 |
|--------|------|--------|----------|------|
| `model` | string | 全局 model | `"提供商/模型名"` 格式 | 该智能体使用的模型 |
| `variant` | string | - | 模型定义的变体名称 | 默认模型变体（仅当使用智能体配置的模型时生效） |
| `temperature` | number | 模型默认值 | `0-2`（实际取决于模型） | 生成随机性，值越高越随机 |
| `top_p` | number | 模型默认值 | `0-1` | 核采样参数，控制词汇多样性 |
| `prompt` | string | - | 任意文本 | 智能体的系统提示词 |
| `steps` | integer | 模型相关 | `>0`, 最大 `9007199254740991` | 强制文本响应前的最大迭代次数 |
| `maxSteps` | integer | - | `>0`（已弃用） | 同 `steps`，已弃用，使用 `steps` 替代 |
| `disable` | boolean | `false` | `true`, `false` | 是否禁用该智能体 |
| `description` | string | - | 任意文本 | 智能体用途描述（用于 @ 菜单） |
| `mode` | string | `"subagent"` | `"subagent"`, `"primary"`, `"all"` | 智能体类型：子智能体/主智能体/两者 |
| `hidden` | boolean | `false` | `true`, `false` | 是否从 @ 自动补全菜单隐藏（仅 mode: subagent） |
| `color` | string | - | Hex 颜色（如 `#FF5733`）或主题色（`primary`, `secondary`, `accent`, `success`, `warning`, `error`, `info`） | 界面中智能体的显示颜色 |
| `options` | object | `{}` | 任意 JSON 对象 | 智能体特定选项（扩展用） |
| `permission` | object | 继承全局 | 见 Permission 配置表 | 该智能体的权限配置 |

### 预定义智能体
配置中可覆盖以下内置智能体：`plan`, `build`, `general`, `explore`, `scout`, `title`, `summary`, `compaction`

## permission 权限配置字段

### 简单模式
| 值 | 说明 |
|----|------|
| `"ask"` | 每次操作都询问用户 |
| `"allow"` | 自动允许所有操作 |
| `"deny"` | 自动拒绝所有操作 |

### 对象模式（权限类型）
| 权限键 | 类型 | 取值范围 | 说明 |
|--------|------|----------|------|
| `read` | PermissionRule | 见下方规则表 | 文件读取权限 |
| `edit` | PermissionRule | 见下方规则表 | 文件编辑权限 |
| `glob` | PermissionRule | 见下方规则表 | 文件模式匹配权限 |
| `grep` | PermissionRule | 见下方规则表 | 内容搜索权限 |
| `list` | PermissionRule | 见下方规则表 | 目录列表权限 |
| `bash` | PermissionRule | 见下方规则表 | Shell 命令执行权限 |
| `task` | PermissionRule | 见下方规则表 | 子任务调用权限 |
| `external_directory` | PermissionRule | 见下方规则表 | 工作区外目录访问权限 |
| `todowrite` | PermissionAction | `"ask"`, `"allow"`, `"deny"` | TODO 列表写入权限 |
| `question` | PermissionAction | `"ask"`, `"allow"`, `"deny"` | 向用户提问权限 |
| `webfetch` | PermissionAction | `"ask"`, `"allow"`, `"deny"` | 网页抓取权限 |
| `websearch` | PermissionAction | `"ask"`, `"allow"`, `"deny"` | 网络搜索权限 |
| `repo_clone` | PermissionRule | 见下方规则表 | 克隆仓库权限 |
| `repo_overview` | PermissionRule | 见下方规则表 | 仓库概览查看权限 |
| `lsp` | PermissionRule | 见下方规则表 | 语言服务器协议调用权限 |
| `doom_loop` | PermissionAction | `"ask"`, `"allow"`, `"deny"` | 避免循环调用权限 |
| `skill` | PermissionRule | 见下方规则表 | 技能调用权限 |

### PermissionRule 取值范围
| 值类型 | 格式 | 示例 | 说明 |
|--------|------|------|------|
| 简单动作 | `"ask"`, `"allow"`, `"deny"` | `"allow"` | 全局应用该动作 |
| 对象映射 | `{ "模式": "动作" }` | `{ "*.js": "allow", "*.env": "deny" }` | 按模式匹配应用不同动作 |

## provider 模型提供商配置字段

| 字段名 | 类型 | 默认值 | 取值范围 | 说明 |
|--------|------|--------|----------|------|
| `name` | string | - | 任意字符串 | 提供商显示名称 |
| `api` | string | - | `"openai"`, `"anthropic"`, `"google"`, `"groq"`, `"mistral"`, `"ollama"`, `"custom"` 等 | API 类型标识 |
| `env` | array | `[]` | 环境变量名数组 | 需要读取的环境变量列表 |
| `id` | string | - | 唯一标识符 | 提供商唯一 ID |
| `npm` | string | - | npm 包名 | 提供商对应的 npm 包 |
| `whitelist` | array | `[]` | 模型 ID 数组 | 仅允许的模型列表（白名单） |
| `blacklist` | array | `[]` | 模型 ID 数组 | 禁止的模型列表（黑名单） |

### provider.options 配置
| 字段名 | 类型 | 默认值 | 取值范围 | 说明 |
|--------|------|--------|----------|------|
| `apiKey` | string | - | API 密钥字符串 | API 密钥（实际建议用环境变量） |
| `baseURL` | string | - | 有效 URL | API 基础 URL |
| `enterpriseUrl` | string | - | 有效 URL | GitHub Enterprise URL（copilot 认证用） |
| `setCacheKey` | boolean | `false` | `true`, `false` | 是否启用 promptCacheKey |
| `timeout` | integer/false | - | `>0` 或 `false` | 完整请求超时时间（毫秒），`false` 禁用 |
| `headerTimeout` | integer/false | - | `>0` 或 `false` | 等待响应头超时（毫秒），`false` 禁用 |
| `chunkTimeout` | integer | - | `>0` | 流式传输块间超时（毫秒） |

### provider.models 模型配置
| 字段名 | 类型 | 取值范围 | 说明 |
|--------|------|----------|------|
| `id` | string | 任意字符串 | 模型唯一标识符 |
| `name` | string | 任意字符串 | 模型显示名称 |
| `family` | string | 如 `"gpt-4"`, `"claude-3"` 等 | 模型系列 |
| `release_date` | string | ISO 日期格式 | 发布日期 |
| `attachment` | boolean | `true`, `false` | 是否支持附件 |
| `reasoning` | boolean | `true`, `false` | 是否支持推理过程 |
| `temperature` | boolean | `true`, `false` | 是否支持 temperature 参数 |
| `tool_call` | boolean | `true`, `false` | 是否支持工具调用 |
| `interleaved` | boolean/object | `true` 或 `{ "field": "reasoning_content" \| "reasoning_details" }` | 是否交错推理内容 |
| `experimental` | boolean | `true`, `false` | 是否为实验性模型 |
| `status` | string | `"alpha"`, `"beta"`, `"deprecated"`, `"active"` | 模型状态 |

#### 模型 cost（成本）配置
| 字段名 | 类型 | 说明 |

| `cache_write` | number | 缓存写入成本（可选） |

#### 模型 limit（限制）配置

## mcp MCP 服务器配置

### McpLocalConfig（本地类型）
| 字段名 | 类型 | 必填 | 取值范围 | 说明 |
|--------|------|------|----------|------|
| `type` | string | ✓ | `"local"` | 连接类型 |
| `command` | array | ✓ | 字符串数组（命令+参数） | 启动 MCP 服务器的命令 |
| `enabled` | boolean | - | `true`, `false` | 是否启用（默认 true） |
| `environment` | object | - | `{ "KEY": "value" }` | 环境变量 |
| `timeout` | integer | - | `>0`, 最大 `9007199254740991` | 请求超时（毫秒，默认 5000） |

### McpRemoteConfig（远程类型）
| 字段名 | 类型 | 必填 | 取值范围 | 说明 |
|--------|------|------|----------|------|
| `type` | string | ✓ | `"remote"` | 连接类型 |
| `url` | string | ✓ | 有效 URL | 远程 MCP 服务器地址 |
| `enabled` | boolean | - | `true`, `false` | 是否启用（默认 true） |
| `headers` | object | - | `{ "Header": "value" }` | 自定义请求头 |
| `oauth` | object/boolean | - | McpOAuthConfig 或 `false` | OAuth 配置，`false` 禁用自动检测 |
| `timeout` | integer | - | `>0`, 最大 `9007199254740991` | 请求超时（毫秒，默认 5000） |

### McpOAuthConfig（OAuth 配置）
| 字段名 | 类型 | 默认值 | 取值范围 | 说明 |
|--------|------|--------|----------|------|
| `clientId` | string | - | 任意字符串 | OAuth 客户端 ID |
| `clientSecret` | string | - | 任意字符串 | OAuth 客户端密钥 |
| `scope` | string | - | 空格分隔的权限范围 | OAuth 请求范围 |
| `callbackPort` | integer | `19876` | `1-65535` | 本地回调服务器端口 |
| `redirectUri` | string | `http://127.0.0.1:19876/mcp/oauth/callback` | 有效 URL | OAuth 重定向 URI |

## 常用模式速查

### 权限配置模式
```json
// 全局单一动作
"permission": "ask"

// 按操作分类
"permission": {
  "read": "allow",
  "edit": "ask",
  "bash": {
    "npm *": "allow",
    "rm *": "deny"
  }
}
```

### MCP 服务器模式
```json
// 本地进程
"mcp": {
  "my-server": {
    "type": "local",
    "command": ["node", "server.js"]
  }
}

// 远程服务
"mcp": {
  "api": {
    "type": "remote",
    "url": "https://api.example.com/mcp"
  }
}
```

### LSP 配置模式
```json
// 启用内置
"lsp": true

// 自定义
"lsp": {
  "typescript": {
    "command": ["typescript-language-server", "--stdio"],
    "extensions": [".ts", ".tsx"]
  }
}
```

此表格涵盖了 `config.json` 中所有定义字段的类型、取值范围和说明，可作为配置时的快速参考手册。
